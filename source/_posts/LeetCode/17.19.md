有些数的素因子只有 3，5，7，请设计一个算法找出第 k 个数。注意，不是必须有这些素因子，而是必须不包含其他的素因子。例如，前几个数按顺序应该是 1，3，5，7，9，15，21。

### 示例 1:
```
输入: k = 5
输出: 9
```
<details><summary>查看代码</summary><pre><code>
class Solution {
public:
    int getKthMagicNumber(int k) {
        vector<int> dp(k + 1);
        int p3 = 1, p5 = 1, p7 = 1;
        dp[1] = 1;
        for (int i = 2; i <= k; i++) {
            int a = dp[p3] * 3;
            int b = dp[p5] * 5;
            int c = dp[p7] * 7;
            dp[i] = min(a, min(b, c));
            if (dp[i] == a) p3++;
            if (dp[i] == b) p5++;
            if (dp[i] == c) p7++;
        }

        return dp[k];
    }
};
</code></pre></details>
