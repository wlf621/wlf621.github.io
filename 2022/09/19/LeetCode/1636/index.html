1636

Author: Wlf Date: 九月 19, 2022  22:30:22 Category: leetcode

给你一个整数数组 nums ，请你将数组按照每个值的频率 升序 排序。如果有多个值的频率相同，请你按照数值本身将它们 降序 排序。

请你返回排序后的数组。

示例 1:

查看代码

class Solution {
public:
    vector frequencySort(vector& nums) {
        unordered_map numsMap;
        for (auto num : nums) {
            numsMap[num]++;
        }
        sort(nums.begin(), nums.end(),
            [&numsMap](auto a, auto b) {
                if (numsMap[a] == numsMap[b]) {
                    return a > b;
                }
                return numsMap[a] < numsMap[b];
            });
        return nums;
    }
};
