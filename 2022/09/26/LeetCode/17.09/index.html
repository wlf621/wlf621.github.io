给定一个数组，包含从 1 到 N 所有的整数，但其中缺了两个数字。你能在 O(N) 时间内只用 O(1) 的空间找到它们吗？

以任意顺序返回这两个数字均可。

    // 按照x比特位是否为1分为两类，出现过两次相同为一类，num1与num2分别在两类，全部异或求出num1、num2

    for (int i = 0; i < nums.size(); i++) {
