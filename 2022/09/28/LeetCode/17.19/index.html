17.09

Author: Wlf Date: 九月 28, 2022  22:30:22 Category: leetcode

有些数的素因子只有 3，5，7，请设计一个算法找出第 k 个数。注意，不是必须有这些素因子，而是必须不包含其他的素因子。例如，前几个数按顺序应该是 1，3，5，7，9，15，21。

示例 1:

查看代码

class Solution {
public:
    int getKthMagicNumber(int k) {
        vector dp(k + 1);
        int p3 = 1, p5 = 1, p7 = 1;
        dp[1] = 1;
        for (int i = 2; i <= k; i++) { int a="dp[p3]" * 3; b="dp[p5]" 5; c="dp[p7]" 7; dp[i]="min(a," min(b, c)); if (dp[i]="=" a) p3++; b) p5++; c) p7++; } 
    return dp[k];
}



};
