940. 不同的子序列 II

Author: Wlf Date: 十月 14, 2022  22:18:07 Category: leetcode

给定一个字符串 s，计算 s 的 不同非空子序列 的个数。因为结果可能很大，所以返回答案需要对 10^9 + 7 取余 。

字符串的 子序列 是经由原字符串删除一些（也可能不删除）字符但不改变剩余字符相对位置的一个新字符串。

  • 例如，”ace” 是 “abcde” 的一个子序列，但 “aec” 不是。

  • 1 <= s.length <= 2000

class Solution {
public:
    int distinctSubseqII(string s) {
        int m = (int)1e9 + 7, ans = 0;
        vector> dp(s.size() + 1, vector(26, 0));
        for (int i = 0; i < s.size(); i++) {
            int c = s[i] - 'a';
            for (int j = 0; j < 26; j++) {
                if (c == j) {
                    int cur = 1;
                    for (int k = 0; k < 26; k++) {
                        cur = (cur + dp[i][k]) % m;
                    }
                    dp[i + 1][j] = cur;
                }
                else {
                    dp[i + 1][j] = dp[i][j];
                }
            }
        }
        for (int k = 0; k < 26; k++) {
            ans = (ans + dp[s.size()][k]) % m;
        }
        return ans;
    }
};
